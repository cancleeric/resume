HTML 產製流程

.md (source)
   ↓
build.sh
   ↓
pandoc -f gfm -t html5
   --template=template.html5
   --metadata-file=presets/<role>.yaml   ← 6 特化版各自 preset
   --metadata title="..."
   ↓
.html (output) — 共用 resume-style.css

元件職責

檔案 職責
template.html5 pandoc HTML 模板（topbar + sidebar + main + footer）
presets/*.yaml 各特化版 sidebar 數據 / 標籤 / 副標 metadata
resume-style.css 雙模樣式（螢幕 104 風 + 印刷 A4 ATS）
build.sh 批次建置（單檔 / 全 build）
assets/avatar.jpg 大頭照（沒檔則 fallback「王」字）

新增特化版 SOP

要新增 resume_xyz.md（例如 resume_devops.md）：

  1. 建 resume_xyz.md，header 第一行 # 王英豪 - XXX
  2. 建 presets/xyz.yaml（含 sidebar_summary / stats[] / sidebar_tags[]）
  3. build.sh preset_for() case 加一行 resume_xyz.md) echo "presets/xyz.yaml" ;;
  4. template.html5 topbar nav 加 <a href="resume_xyz.html">XXX</a>
  5. README.md 履歷導覽表加一列
  6. INSTRUCTIONS.md 1 主版 + N 特化版 表加一列
  7. ./build.sh 全 build 驗證
  8. 開 resume_xyz.html CEO 親驗

常用指令

./build.sh                    # 全 build
./build.sh resume_dotnet.md   # 單檔 build
open resume.html              # 開瀏覽器看
